# matlab_linter
A MATLAB Linter for Ann and H.J. Smead Undergrad Aerospace Homework Assignments

Please see matlab_linter.py for license.

Also in this repository should be matlab_linter.exe which is a pyinstaller binary for Windows so that Windows users do not require a python installation to run the program.

To learn how to use the program works, in a terminal just type python matlab_linter.py -h, or read the python file for more details
